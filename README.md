# OpenPocket

A privacy-first, fully offline AI note-taking system with a custom BLE wearable.

- **Zero cloud** — all AI inference runs on-device using Liquid AI's LEAP SDK
- **Custom hardware** — ESP32-S3 + MEMS mic wearable (BOM < $15)
- **Flutter iOS app** — forked from Omi, with Firebase/cloud replaced by local models

See [project_plan.md](project_plan.md) for the full architecture and build plan.
